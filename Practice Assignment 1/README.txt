Завдання 1
Створила базу даних для управління лікарнею. Вона складається з 5 табличок - пацієнти, лікарі, діагнози, процедури та записи.
У кожній є унікальні ID для пошуку певного типу за ним і зв'язки з іншими таблицями. Також реалізовано CTE для фільтрації вартості процедур - дешева або дорога.
Через select я обираю колонки ім'я пацієнта та лікаря, діагноз, вартість за фільтрацією та загальну вартість процедури.
Потім об'єдную всі ключові значення та виводжу результати записів, де вартість процедури - дорога. Вкінці виведення по спаданню вартості.
